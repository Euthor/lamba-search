# lambda-search
Fuzzy searches as serverless lambdas

## Scripts

## Deploy

```bash
$ serverless deploy
```

### Run production

```bash
$ serverless invoke --function search --data "John Doe"
```

### Run Local

For invoking searches locally

```bash
$ serverless invoke local --function search --data "John Doe"
```
